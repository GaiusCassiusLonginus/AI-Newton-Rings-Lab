# Reproducibility notes

This repository is intended as a minimal reproducibility package associated with the manuscript on an AI-assisted Newton's rings laboratory.

## Included in the current release

The current release includes:

- representative Python scripts for ring detection, manual measurement, and prediction-related workflow
- sample Newton's rings images
- a small sample measurement table
- example output files
- basic repository structure for code, data, results, configuration, and documentation

## Scope of the current release

This release is intended to support:

- inspection of the general workflow
- basic reproducibility checks
- peer-review access to representative materials
- classroom adaptation at a conceptual and procedural level

## Not yet included as a full release

The current version is not yet a complete archival release of all classroom materials and all raw datasets.

Future updates may expand:

- the size of the sample dataset
- configuration records
- uncertainty-analysis documents
- teaching implementation materials
- additional processed outputs

## Use of the repository

Users are encouraged to inspect the scripts, sample images, measurement table, and example outputs together rather than treating any single file as a complete reproduction of the full study.
